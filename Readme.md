## Clone the repository to execute it on your local machine.


```bash
  git clone https://github.com/GauravOOO2/Coding-assessment-l2.git
```

Go to the project directory

```bash
  cd Coding-assessment-l2
```

Install dependencies

```bash
  npm install
```
Run Frontend


```bash
  npm install
  npm start
```
